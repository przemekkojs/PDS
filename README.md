# PDS - Przetwarzanie Danych Strumieniowych
Projekt

Przygotował: Przemysław Kojs

## Aplikacja
Jako projekt stworzona została aplikacja służąca do predykcji czasu przejazdu oraz ceny dla taksówek w Nowym Jorku (zbiór NYC Taxi Cabs). Dotyczy jedynie żółtych taksówek (yellow cabs).

## Uruchomienie
Aby uruchomić projekt z użyciem `uvicorn` (po sklonowaniu repozytorium)
1. `pip install -r requirements.txt`
2. `cd app/backend/app`
3. `uvicorn app:main`

A następnie można uruchomić plik `frontent/index.html` z użyciem rozszerzenia LiveServer wbudowanego w VS Code.

## Więcej informacji
Znajduje się w pliku `Projekt.ipynb`, który jest _brudnopisem_ projekt oraz w pliku `Rapord.pdf`, który jest jego głównym raportem, stworzonym na potrzeby kursu. Możliwe również, że tego pliku w repozytorium nie ma (jeżeli zapomniałem go wrzucić).
